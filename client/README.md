# 智邻便民客户端（登录版）

## 功能概览
- 手机号 + 密码注册
- 手机号 + 密码登录
- 登录态持久化（`localStorage`）
- 个人任务查询 / 新建 / 删除（基于 `/api/me/tasks`）
- 退出登录

## 运行方式
1. 在项目根目录启动服务：
   - `npm start`
2. 浏览器访问：
   - `http://localhost:3000/`
