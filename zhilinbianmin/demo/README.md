# 智邻便民前端演示页面

## 运行方式
1. 直接双击 `index.html` 用浏览器打开（建议 Chrome / Edge 最新版）。
2. 若需要避免本地跨域限制，使用任意静态服务启动目录即可。

## 外部依赖
- `ECharts 5.x`（通过 CDN 引入）
- `Google Fonts`（Noto Sans SC / Noto Serif SC，通过 CSS `@import` 引入）

## 目录结构
```
demo/
  index.html
  README.md
  assets/
    css/
      style.css

      main.js
    images/

```
